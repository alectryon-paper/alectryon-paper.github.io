Booleans

The built-in Boolean type has two constructors:

Inductive bool : Set :=
    true : bool | false : bool

Induction principles

Coq automatically defines an induction principle:

bool_ind:
  ∀ P : bool → Prop,
    P true → P false → ∀ b : bool, P b
